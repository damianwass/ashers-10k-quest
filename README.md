# Asher's 10K Quest Tracker

This is a static, no-build version of the tracker.

## Deploy on Vercel

Upload the contents of this folder to GitHub.

Important: there is no package.json. That is intentional.
Vercel should deploy it as a static site with no npm install and no build step.

## Data storage

Run data is stored in the browser using localStorage on the device where the app is used.
