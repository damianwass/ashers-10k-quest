# Asher's 10K Quest Tracker

Static no-build version.

## v5 updates

- Moved progress bars above This Week’s Quest
- Added weekly power-up message under the banner
- Weekly message changes when the selected week changes
- Added badges for run count, consistency streaks, and plan completion percentage
- Keeps v4 workout flow: Not Started, Complete, Incomplete, Skipped

## Deploy on Vercel

Upload only these files to GitHub:

- index.html
- README.md

Vercel settings:

- Framework Preset: Other
- Build Command: blank
- Output Directory: .
- Install Command: blank

## Data storage

Run data is stored in the browser using localStorage on the device where the app is used.
