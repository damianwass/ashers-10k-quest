# Asher's 10K Quest Tracker

Static no-build version.

## v6 updates

- Moves the weekly Power-Up Message into the This Week's Quest card
- Message now appears above the week selector
- Removes the separate Power-Up Message widget

## Deploy on Vercel

Upload only these files to GitHub:

- index.html
- README.md

Vercel settings:

- Framework Preset: Other
- Build Command: blank
- Output Directory: .
- Install Command: blank

## Data storage

Run data is stored in the browser using localStorage on the device where the app is used.
