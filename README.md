# Asher's 10K Quest Tracker

Static no-build version.

## v8 updates

- Removes the duplicate standalone Power Up widget
- Keeps the Power Up message only inside This Week's Quest, below the week selector

## Deploy on Vercel

Upload only these files to GitHub:

- index.html
- README.md

Vercel settings:

- Framework Preset: Other
- Build Command: blank
- Output Directory: .
- Install Command: blank

## Data storage

Run data is stored in the browser using localStorage on the device where the app is used.
